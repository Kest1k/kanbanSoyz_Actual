# Канбан-доска для Союз-PLM

Инструмент управления задачами, встроенный в BIS Союз-PLM как дополнительный экран. Работает без установки стороннего ПО: в систему загружаются только серверные C#-скрипты, HTML-шаблон и клиентские CSS/JS-ресурсы.

## Возможности

- **Drag & drop** задач между колонками: Надо сделать / В работе / Ожидание / Готово
- **Карточка задачи**: описание, срок, приоритет, исполнитель, история изменений, чат, вложения
- **Ролевая иерархия по `Commission` + `Context`**:
  сотрудник видит только свои задачи,
  ведущий инженер (`NameKey` начинается с `ved`) и начальник сектора работают в рамках своего сектора,
  начальник отделения видит отделение,
  администратор видит всю организацию
- **Серверная валидация назначения**: задачу нельзя выдать вне разрешённого scope даже при обходе UI
- **Групповая постановка**: создание отдельной задачи сразу нескольким исполнителям
- **Отчёты**: выполнено / просрочено / в работе по исполнителю за период
- **Вложения**: прикрепление объектов и контейнеров PLM к задаче
- **Чат**: комментарии внутри карточки задачи
- **Отдельная кнопка открытия доски**: optional-скрипт `kanbanButton.cs` для команды, открывающей экран `UI\Screens\myKanbanTest`
- **IE11-совместимость**: работает в WebBrowser ActiveX control (BIS v3)

## Стек

| Слой | Технология |
|------|-----------|
| Backend | C# scripting в Союз-PLM BIS v3 |
| Frontend | HTML + CSS + JavaScript (ES5, IE11) |
| Bridge | `window.external.InvokeTemplate()` |
| Шаблонизатор | Liquid / DotLiquid (серверный рендер HTML) |

## Структура

```text
scripts/                 ← файлы для загрузки в Союз
  SOYUZ_UPLOAD_KanbanScreen_script.cs   серверный C# скрипт экрана
  SOYUZ_UPLOAD_KanbanTask_script.cs     скрипт шаблона задачи
  kanbanButton.cs                       optional-скрипт команды открытия доски
  KanbanBoard_HTML.html                 HTML-шаблон экрана
  kanban.css                            стили (→ CSS/ на сервере)
  kanban.js                             логика (→ JS/ на сервере)

docs/                    ← техническая документация
  00–05_*.md             модульная документация по компонентам
  HTML/                  HTML-версия документации
```

## Развёртывание

1. Создать справочники `Ref_KanbanStatus`, `Ref_KanbanPriority` и шаблон `KanbanTask` с нужными атрибутами.
2. Скопировать `kanban.css` → `CSS/`, `kanban.js` → `JS/` на сервер.
3. Загрузить `SOYUZ_UPLOAD_KanbanScreen_script.cs` в Script экрана.
4. Загрузить `KanbanBoard_HTML.html` в HTML/Viewer экрана.
5. При необходимости загрузить `kanbanButton.cs` как скрипт команды кнопки открытия доски.
6. Проверить `<base href>` в шаблоне — он должен совпадать с адресом сервера.

Или использовать файл `Kanban Конфигурация-1.0.0.2.pmszcfg` / `Kanban Конфигурация-1.0.0.3.pmszcfg` для импорта конфигурации.
