## Структура проекта

```
kanbanSoyz_Actual/
├── scripts/                    ← рабочие скрипты для загрузки в Союз
│   ├── SOYUZ_UPLOAD_KanbanScreen_script.cs   серверный C# скрипт экрана
│   ├── SOYUZ_UPLOAD_KanbanTask_script.cs     скрипт шаблона задачи
│   ├── KanbanBoard_HTML.html                 HTML-шаблон экрана
│   ├── kanban.css                            стили (→ CSS/ на сервере)
│   └── kanban.js                             логика (→ JS/ на сервере)
├── docs/                       ← техническая документация
│   ├── 00–05_*.md              модульная документация
│   └── HTML/                   HTML-версия документации
├── AGENTS.md                   ← этот файл
├── FULL_CONTEXT_OF_PROJECT.md  ← полный контекст для агентов
├── README.md                   ← обзор проекта
└── Kanban Конфигурация-*.pmszcfg  ← экспорт конфигурации PLM
```

История разработки (iterations, промежуточные версии, оптимизационные планы) находится в отдельном репозитории: [kanbanSoyz2](https://github.com/Kest1k/kanbanSoyz2).
