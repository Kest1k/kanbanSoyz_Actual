# Шаг 08 — Презентационные материалы

> Отдельный пакет материалов. Этот шаг не встраивается в runtime, а помогает показать текущее состояние проекта руководству.

## Назначение

Подготовить презентацию о возможностях и пользе канбан-доски внутри Союз-PLM и показать уже внедрённое состояние шагов 1-10.

## Что стоит показывать по текущему runtime

- 4-колоночную доску и drag-and-drop
- карточку задачи
- уведомления при назначении
- ролевую иерархию
- отчёты
- вложения объектов и контейнеров
- комментарии в карточке задачи

## Файлы пакета

- `presentation_outline.md` — структура и тексты слайдов
- `talking_points.md` — тезисы для выступления
- `ИНСТРУКЦИЯ.md` — краткий порядок подготовки материалов
- `screenshots/` — папка для скриншотов

## Статус

- пакет актуален как основа для презентации
- источником истины по фактическому функционалу остаются `CURRENT_*`
