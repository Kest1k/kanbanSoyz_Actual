# Шаг 08 — Презентация для руководства

> Отдельный пакет материалов. Источником истины по функционалу остаются `CURRENT_KanbanScreen_script.cs` и `CURRENT_KanbanBoard_HTML.html`.

## Зависимости
- Желательно готовить после сверки актуального runtime шагов 1-10
- Текст и структуру можно поддерживать параллельно

## Что подготовить
1. Текст слайдов (`presentation_outline.md`)
2. Тезисы для выступления (`talking_points.md`)
3. Скриншоты работающей доски (папка `screenshots/`)

## Порядок действий

### Шаг 1: Подготовить скриншоты
После сверки актуального runtime сделай скриншоты:
- Общий вид доски с задачами в разных колонках
- Создание задачи (панель создания)
- Перетаскивание карточки (drag-and-drop)
- Карточка задачи (шаг 07)
- Отчёт по задачам (шаг 06)
- Переключатель «Мои / Сектор / Отделение» (шаг 05)
- Уведомление о назначении (шаг 04)
- Вложения и комментарии в карточке задачи (шаги 09-10)

### Шаг 2: Создать презентацию
Используй `presentation_outline.md` как основу для слайдов.
Формат — PowerPoint или Google Slides.

### Шаг 3: Подготовиться к выступлению
Используй `talking_points.md` для репетиции.
Рекомендуемое время: 10-15 минут.
