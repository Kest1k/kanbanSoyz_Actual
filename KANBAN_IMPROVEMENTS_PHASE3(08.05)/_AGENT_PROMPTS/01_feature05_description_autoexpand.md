Ты работаешь в репозитории Soyuz-PLM Kanban на Windows. Платформа фронта: WebBrowser-контрол Trident (IE11 в режиме edge через X-UA-Compatible).

ЖЁСТКИЕ ПРАВИЛА КОДА:
- Только ES5 в kanban.js: var/function. Никаких const/let/=>/template literals/optional chaining/spread.
- Никаких CSS-переменных var(--x) — IE11 их не поддерживает. Использовать готовые цвета.
- Никаких новых зависимостей, никакого npm, никакого webpack, никаких сторонних библиотек.
- C# — стиль .NET Framework, как в существующем SOYUZ_UPLOAD_KanbanScreen_script.cs.
- JS-вызов сервера: window.external.InvokeTemplate("Method", "args|разделённые|пайпом").
- Текст в JSON-полях защищать через text.replace(/\|/g, " ") на клиенте.
- Существующие имена функций/переменных НЕ переименовывать (другие файлы зависят).
