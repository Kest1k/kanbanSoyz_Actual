# Smoke-тест 04 — Режим «Выданные мной»

Длительность: ~5 мин. Понадобится учётка руководителя (headOfSector / headOfDept / admin) и хотя бы один подчинённый.

## Подготовка

1. Залогиниться руководителем.
2. Создать 2 задачи на себя («Тест 04 — Сам себе #1», «Тест 04 — Сам себе #2»).
3. Создать 2 задачи на подчинённого («Тест 04 — Подчинённому #1», «Тест 04 — Подчинённому #2»).
4. Также убедиться, что в системе есть задачи, выданные **мне** другим руководителем (либо создать админом).

## 1. Базовый сценарий

- [ ] В шапке появилась кнопка «Выданные мной».
- [ ] Изначально активна кнопка «Мои задачи» (синяя).
- [ ] Клик по «Выданные мной»:
  - Доска показывает 4 моих задачи (2 себе + 2 подчинённому).
  - **Не показывает** задачи, выданные мне кем-то другим.
  - Селекты сектора / сотрудника сброшены.
  - Кнопка «Выданные мной» подсветилась синим, «Мои задачи» / «Все задачи» — серые.

## 2. Возврат к «Мои задачи»

- [ ] Клик по «Мои задачи»:
  - Доска показывает только задачи, где Я Assignee (2 на себя + те, что выданы мне другим).
  - Не показывает задачи, выданные мной подчинённому (там Assignee = подчинённый).
  - Кнопка «Мои задачи» подсветилась.

## 3. Возврат к «Все задачи»

- [ ] Клик по «Все задачи» / «Всё отделение» / «Весь сектор» (название зависит от роли):
  - Доска показывает все задачи скоупа (мои + подчинённых).
  - Кнопка подсветилась.

## 4. Фильтр периода + «Выданные мной»

- [ ] Клик по «Период» → «На этой неделе» → Применить.
- [ ] Клик по «Выданные мной».
- [ ] Доска показывает только задачи, **выданные мной** + **созданные на этой неделе**.
- [ ] Сбросить период → видно все мои выданные.

## 5. Перезагрузка страницы

- [ ] Будучи в режиме «Выданные мной» — нажать F5 / закрыть-открыть Soyuz клиент.
- [ ] При повторном открытии Канбан-доски кнопка «Выданные мной» снова активна, доска отфильтрована.

## 6. Регрессия

- [ ] Обычный сотрудник (regular): кнопка «Выданные мной» **не отображается**.
- [ ] Руководитель в режиме «Выданные мной» **может** двойным кликом открыть карточку задачи и редактировать.
- [ ] Создать новую задачу подчинённому из режима «Выданные мной» — задача появится на доске сразу после создания (счётчик колонки увеличится).
- [ ] Удалить задачу (которую я создал) — пропадёт с доски.

## 7. Legacy-задачи

- [ ] Если есть очень старые задачи без атрибута Creator (созданные до Phase 2): они **не попадут** в «Выданные мной». Это ожидаемо. (Альтернатива: если на стенде нужен бэкфилл — это отдельная задача администратора, не входит в эту доработку.)

## 8. Группы-задачи

- [ ] Создать групповую задачу (один Title, несколько исполнителей сразу).
- [ ] В режиме «Выданные мной» все экземпляры группы видны на доске у меня (по одному на каждого исполнителя).
