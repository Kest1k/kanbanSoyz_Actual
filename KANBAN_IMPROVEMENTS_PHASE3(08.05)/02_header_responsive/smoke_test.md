# Smoke-тест 02 — Адаптация шапки при сжатии окна

Длительность: ~3 мин. Понадобится мышь (для перетаскивания границ окна).

## Подготовка

1. Открыть Канбан-доску в обычной ширине окна (≥ 1400 px).
2. Залогиниться руководителем (любой не-обычный сотрудник), чтобы была видна панель иерархии (Мои задачи / Все задачи / селекты).

## 1. Полная ширина — регрессия

- [ ] Слева брендинг «Канбан-доска».
- [ ] Далее ссылки «Новая задача / Обновить / Отчёт».
- [ ] Далее панель иерархии — кнопки «Мои задачи / Все задачи» и селекты.
- [ ] Далее кнопка «Период».
- [ ] Справа — «Что нового» (зелёная) и «Справка» (синяя).
- [ ] Никаких визуальных нахлёстов.

## 2. Сжатие до средней ширины (~1100 px)

- [ ] Уменьшить окно до ~1100 px (или половины экрана).
- [ ] Кнопки «Что нового» и «Справка» **уезжают на новую строку** и прижимаются к правому краю.
- [ ] Левая часть (нав + панели) остаётся на верхней строке.
- [ ] Никаких пересечений.

## 3. Сжатие до узкой ширины (~760 px)

- [ ] Уменьшить окно до ~760 px.
- [ ] Список ссылок «Новая задача / Обновить / Отчёт» **переносится на новую строку**.
- [ ] Панель иерархии и фильтр периода тоже переносятся.
- [ ] Все элементы видны и кликаются. Никакие кнопки не наезжают.

## 4. Сжатие до минимума (~600 px)

- [ ] Уменьшить до 600 px. Шапка многострочная, но **все элементы доступны**.
- [ ] Если что-то «вылезло» вправо за окно — это баг, надо обернуть.

## 5. Функциональная регрессия

- [ ] Кнопка «Новая задача» открывает панель создания.
- [ ] Кнопка «Обновить» обновляет доску.
- [ ] Кнопка «Период» открывает выпадашку, она якорится **под** кнопкой Период (а не уезжает в угол).
- [ ] Кнопки «Мои задачи / Все задачи» переключают режим.
- [ ] Селекты иерархии открываются и срабатывают.
- [ ] Кнопка «Что нового» открывает модалку.
- [ ] Кнопка «Справка» открывает справку.

## 6. Регрессия для обычного сотрудника

- [ ] Зайти под пользователем без прав (regular). Панель иерархии не отображается.
- [ ] Шапка содержит только брендинг + ссылки + Период + Что нового + Справка.
- [ ] Адаптация работает: при сжатии — корректно переносится.
